# 搜尋字詞分析師 (Search Query Analyst)

本技能指導如何分析真實搜尋字詞報告 (Search Query Report)，找出無效曝光與非意向點擊，並規劃否定關鍵字清單以減少不必要的廣告費支出。

## 1. 搜尋字詞診斷與分類 (Search Query Audit)
- **篩選低效字詞**：分析展現量高但無轉換的搜尋字詞，特別是點擊次數達臨界值（如超過平均 CPA 的花費）卻未產生轉換的字詞。
- **意圖分析**：區分資訊型（如「如何...」）、商業調查型（如「最佳機車推薦」）與交易型（如「租機車 台北 便宜」）搜尋字詞，排除不具轉換價值的字詞。

## 2. 否定關鍵字清單建置 (Negative Keywords Planning)
- **排除無效關鍵字**：產出否定關鍵字（Negative Keywords）清單。可按以下層級分類：
  - **通用排除字**：免費、求職、新聞、盜版、破解、維修（非租車業務）等。
  - **不相關地理位置**：例如只在台北提供租車，應排除台中、高雄等地區名。
  - **競品字詞**：若策略上不競標競品，應排除同業名稱以避免高昂 CPC。
- **比對模式應用**：正確使用否定廣泛、否定片語與否定精確比對，避免錯誤排除有潛力的流量。
