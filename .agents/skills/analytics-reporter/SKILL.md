# 數據分析與回報專家 (Analytics Reporter)

本技能提供建立行銷成效監控報表與分析框架，幫助團隊及決策者直觀理解廣告花費的投資回報率 (ROI) 與漏斗表現。

## 1. Looker Studio 儀表板架構設計
- **首頁綜觀 (Executive Summary)**：呈現最核心的主指標：總花費、轉換數、平均 CPA、總轉換價值與綜合 ROAS。
- **管道成效分析 (Channel Performance)**：細分 Google Ads, Meta Ads 等各管道的佔比與效率對比。
- **漏斗轉化分析 (Funnel Conversion)**：從廣告曝光 → 點擊 → 網站造訪 → KYC 註冊 → 付款預訂，分析每一步的流失率。

## 2. 成效回報與優化建議框架 (Reporting Framework)
撰寫週報/月報時，採用以下結構：
- **上期成效概述**：用數據說話，指出主要指標相比上期的增長或滑落比例。
- **主要增長點分析 (Wins)**：找出哪些廣告活動、關鍵字或素材表現最好，分析其成功原因。
- **痛點與損失分析 (Losses)**：指出成效不佳的原因（如出價過高、搜尋字詞走偏、金流跳轉失敗）。
- **下一步具體行動 (Next Steps)**：列出 3-5 項具體可執行的優化動作（Action Items）。
