# 付費媒體稽核師 (Paid Media Auditor)

本技能提供全面的廣告帳戶稽核方法，用於在新接管帳戶時快速診斷問題、分析預算效率與評估歷史成效。

## 1. 帳戶結構與設定審查 (Account Structure & Setup Audit)
- **架構評估**：檢查廣告活動 (Campaign) 與廣告群組 (Ad Group) 是否符合 Hagakure 或主題化架構，避免結構過於破碎。
- **預算分佈**：分析預算是否過度分散在低效廣告活動，或預算是否受限 (Budget Constrained)。
- **出價策略**：審查目前的出價模式（如 Target CPA, Maximize Conversions, Manual CPC）是否與業務目標一致。

## 2. 核心成效指標診斷 (Key Performance Indicators)
分析歷史數據（通常為過去 30 到 90 天），診斷以下指標的健康度：
- **點閱率 (CTR)**：評估廣告創意與關鍵字的相關性。
- **轉換率 (CVR)**：評估落地頁 (Landing Page) 的說服力與受眾意向。
- **每次轉換成本 (CPA) & 廣告投資報酬率 (ROAS)**：評估帳戶整體的獲客效率與獲利能力。

## 3. 常見警訊與改進建議
- **重複競價 (Keyword Cannibalization)**：相同帳戶內多個廣告活動競標相同關鍵字，導致內部競爭與花費浪費。
- **低品質分數 (Quality Score)**：分析關鍵字、廣告文案與落地頁的關聯性，找出品質分數低於 5 的關鍵字。
